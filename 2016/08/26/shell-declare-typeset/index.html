命令用途

declare 与 typeset 命令是bash的内建命令，两者是完全一样的，用来声明shell变量，设置变量的属性。

常用命令参数

  1. -r 设置变量为只读

  2. -i 设置变量为整数

  3. -a 设置变量为数组array

  4. -f 如果后面没有参数的话会列出之前脚本定义的所有函数，如果有参数的话列出以参数命名的函数

  5. -x 设置变量在脚本外也可以访问到
